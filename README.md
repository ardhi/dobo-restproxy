Rest API Proxy driver for [Dobo](https://github.com/ardhi/dobo)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install dobo-restproxy
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```dobo-restproxy``` in it
